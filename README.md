# 🏢 회의실 예약 시스템 — 배포 가이드

## 📱 폰 앱처럼 설치하는 방법 (PWA)

이 앱은 **앱스토어 없이** 아이폰·안드로이드 홈화면에 설치할 수 있어요.

---

## 🚀 Step 1. GitHub에 올리기

1. [github.com](https://github.com) 접속 → 회원가입 (무료)
2. 오른쪽 상단 **+** 버튼 → **New repository** 클릭
3. Repository name: `meeting-room-booking`
4. **Create repository** 클릭
5. 화면에 나오는 안내에 따라 이 폴더 파일들을 업로드
   - **"uploading an existing file"** 링크 클릭
   - 이 폴더 안의 모든 파일을 드래그 앤 드롭
   - **Commit changes** 클릭

---

## 🌐 Step 2. Vercel로 배포하기 (무료)

1. [vercel.com](https://vercel.com) 접속
2. **Sign Up** → **Continue with GitHub** 선택
3. **New Project** 클릭
4. 방금 만든 `meeting-room-booking` 저장소 선택 → **Import**
5. 설정 그대로 두고 **Deploy** 클릭
6. 잠시 기다리면 🎉 **배포 완료!**
   - 예시 URL: `https://meeting-room-booking.vercel.app`

---

## 📲 Step 3. 폰 홈화면에 설치하기

### 아이폰 (Safari에서만 가능)
1. Safari로 위 URL 접속
2. 하단 공유 버튼(□↑) 탭
3. **"홈 화면에 추가"** 선택
4. **추가** 탭 → 완료!

### 안드로이드 (Chrome)
1. Chrome으로 URL 접속
2. 우측 상단 점 3개 메뉴 탭
3. **"앱 설치"** 또는 **"홈 화면에 추가"** 선택
4. **설치** 탭 → 완료!
